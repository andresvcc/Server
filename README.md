# Server
**Pour démarrer le client de façon local vous avez besoin de** `npm`.

# Installation
**pour installer:** install.
```bash
$ npm install
```

# Usage
**pour demarrer:** start.
```bash
$ npm start
```

# Usage de develepement 
**pour demarrer:** dev.
```bash
$ npm run dev
```
il sera deploye par default dans le port 4000, http://localhost:4000 :+1:
